Welcom to the Match-Match Game!

How to play

Memory is a counter game in which an object must find pairs.

When the game begins, all pictures are hidden.

To play

  1. Select two cards to try to match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it is the computer turn them
  4. The player that finds all pairs wins!
  5. Have Fun!

Skirt Cards

  • Alphabets
  • Animals
  • Architecture

Game Difficulty

  • Low (5 x 2)
  • Medium (6 x 3)
  • Height (8 x 5)

How to start

  1. Choose skirt cards and game difficulty.
  2. Complete your information on the form.
  3. Click start.
